## 4. Component Specifications

### 4.1 Timeline Component

**Purpose:** Visual representation of daily schedule with tankers as rows and time as columns.

**Features:**
- 30-minute time slots (00:00 - 23:30)
- Horizontal scrolling for time axis
- Fixed tanker column on left
- Trip blocks with color-coding
- Click to select trip
- Drag-and-drop to reassign (future)
- Zoom controls (optional)

**Props:**
```typescript
interface TimelineProps {
  date: Date;
  tankers: Tanker[];
  trips: Trip[];
  onTripClick: (trip: Trip) => void;
  onSlotClick: (tanker: Tanker, time: string) => void;
}
```

### 4.2 Trip Card Component

**Purpose:** Display trip information in the trip list.

**Features:**
- Customer code and name
- Time window
- Tanker and driver assignment
- Status indicator (color badge)
- Action buttons (edit, assign, delete)

**Status Colors:**
- Green: Scheduled (fully assigned)
- Yellow: Unassigned (needs tanker/driver)
- Red: Conflict (overlapping or constraint violation)
- Gray: Cancelled

### 4.3 Driver Schedule Grid Component

**Purpose:** Monthly grid view of driver availability.

**Features:**
- Driver names in first column
- Days as column headers
- Cells showing status with color coding
- Click cell to change status
- Bulk selection for setting patterns
- Export functionality
